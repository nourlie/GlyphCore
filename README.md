# GW2 Cyrillic

### Русификатор Guild Wars 2 — кириллический шрифт + перевод текста

> **Русификатор / руссификатор Guild Wars 2** (русская локализация GW2): рендеринг
> кириллицы в игре + перевод игрового текста на русский. Перевод GW2 на русский язык
> без изменения `Gw2.dat`. *Guild Wars 2 Russian localization / Russifier.*

<img width="237" height="247" alt="image" src="https://github.com/user-attachments/assets/2d33dddc-26aa-4c0f-bf60-b0dcd12c216c" />

Прокси-DLL ранней загрузки, которая заставляет **Guild Wars 2** **рисовать
кириллицу** — буквы, которые штатный клиент показывает пустыми квадратами — и
**переводит игровой текст** на русский. Это проект для доступности / локализации:
он учит живой движок шрифтов рисовать кириллицу и подменяет текст **в памяти**,
**без изменения файла `Gw2.dat` на диске**, поэтому проверки целостности архива
ArenaNet проходят и игра не запускает «починку».

> **Статус:** ✅ Кириллица рендерится в игре без краша (покрыто 136 / 156
> размеров шрифта). ✅ Перевод текста на русский, включая ранее нечитаемые
> зашифрованные (RC4) строки (~70 % текста).

> ⬇️ **Готовые файлы — на странице [Releases](https://github.com/nourlie/gw2-cyrillic/releases/latest).**
> Собирать из исходников не нужно — просто скачайте и положите в папку игры.

> ⚠️ **Готовый перевод в комплект не входит.** Релиз содержит `version.dll`
> (рендеринг кириллицы) и `cyrillic-string-dump-tool.exe`. **Словарь
> `cyrillic_strings.csv` вы наполняете сами** — извлеките английский текст
> dump-tool'ом и переведите (или копите строки в игре через `autocollect`). Без
> заполненного словаря кириллица рисуется, но текст остаётся английским.

## 📦 Установка

1. Скачайте **`version.dll`** из [последнего релиза](https://github.com/nourlie/gw2-cyrillic/releases/latest).
2. Положите его в папку Guild Wars 2 — рядом с `Gw2-64.exe`.
3. Запустите игру. Готово.

```
Guild Wars 2\
├─ Gw2-64.exe
└─ version.dll      ← скачанный файл
```

> Настоящие экспорты `version.dll` форвардятся в подлинную системную `version.dll`
> из `System32` автоматически — отдельная копия `version_orig.dll` не нужна.

> 🔄 **Автообновление.** При запуске `version.dll` проверяет
> [GitHub-релизы](https://github.com/nourlie/gw2-cyrillic/releases) и, если есть
> новее, тихо обновляет себя к следующему запуску (фоном, при наличии сети).
>
> **Откатить:** удалить `version.dll` из папки игры.

### 🅰️ Свой кириллический шрифт (опционально)

По умолчанию используется зашитый **Roboto**. Чтобы рисовать кириллицу **своим**
шрифтом, положите файл **`cyrillic.ttf`** в подпапку **`cyrillic\`** рядом с
`version.dll` — глифы будут отрисованы из него в рантайме под каждый размер.

```
Guild Wars 2\
├─ Gw2-64.exe
├─ version.dll
└─ cyrillic\
   └─ cyrillic.ttf   ← ваш шрифт
```
> Если положить `cyrillic.ttf` / `cyrillic_strings.csv` прямо в папку игры,
> при следующем запуске `version.dll` сама перенесёт их в `cyrillic\`.

> [!IMPORTANT]
> **Шрифт обязан содержать кириллические глифы.** Если в `.ttf` нет кириллицы,
> version.dll **отклонит его** и откатится на зашитый Roboto — без вылета.
> Проверить можно в логе `cyrillic\version_proxy_log.txt`:
> `user TTF accepted` — принят; `user TTF REJECTED: no Cyrillic glyphs` — Roboto.

> [!WARNING]
> **Не доверяйте окну предпросмотра шрифта в Windows** — оно показывает русский
> даже у шрифтов без кириллицы (системный font fallback), а в самом файле букв нет.
> Берите шрифты с реальным покрытием кириллицы:
> [Roboto](https://fonts.google.com/specimen/Roboto),
> [Open Sans](https://fonts.google.com/specimen/Open+Sans),
> [PT Sans](https://fonts.google.com/specimen/PT+Sans),
> [Noto Sans](https://fonts.google.com/noto/specimen/Noto+Sans).

## 📝 Перевод текста игры

Помимо отрисовки кириллицы, `version.dll` **переводит игровой текст** на русский.
Перевод берётся из словаря **`cyrillic\cyrillic_strings.csv`** (колонки
`english,russian`, редактируется в любой таблице). Английская строка заменяется
русской **в памяти**, после проверок целостности — `.dat` не трогается. **Длина
русского не ограничена.** Переводятся в том числе зашифрованные (RC4) строки —
это ~70 % всего текста игры.

```
Guild Wars 2\
├─ version.dll
├─ cyrillic-string-dump-tool.exe   ← инструмент дампа словаря (из релиза)
└─ cyrillic\
   ├─ cyrillic_strings.csv         ← словарь (english,russian) — заполняете русский
   ├─ discovered_strings.csv       ← авто-собираемые новые строки (временный)
   ├─ cyrillic.ttf                 ← свой шрифт (опционально)
   └─ version_proxy_log.txt        ← лог
```

### Получить базовый словарь — `cyrillic-string-dump-tool.exe`

**Незашифрованные** строки (имена предметов и часть UI — это меньшая часть текста)
можно извлечь из `Gw2.dat` офлайн инструментом **`cyrillic-string-dump-tool.exe`**
(скачивается из того же
[релиза](https://github.com/nourlie/gw2-cyrillic/releases/latest)):

1. **Полностью закройте игру** (и лаунчер — ему нужен эксклюзивный доступ к архиву).
2. Положите `cyrillic-string-dump-tool.exe` в папку Guild Wars 2 (рядом с `Gw2.dat`)
   и запустите (двойной клик).
3. Он сканирует архив (несколько минут) и **дописывает** найденные английские строки
   в `cyrillic\cyrillic_strings.csv` (merge — существующие строки и ваши переводы
   сохраняются).

Затем заполняйте колонку `russian`.

> ℹ️ **Зашифрованные (RC4) строки — ~70 % текста — этим инструментом офлайн НЕ
> извлекаются** (для них нужны рантайм-ключи игры). Они попадают в словарь **по
> мере игры** через `autocollect` (см. ниже): `version.dll` декодирует их на лету и
> записывает. Итог: **полный словарь = базовый офлайн-дамп (raw) + автосбор RC4 в
> игре.**

### Автосбор по мере игры

С `autocollect=true` (см. ниже) `version.dll` декодирует каждую новую строку,
которую показывает игра (включая зашифрованные), и пишет её в
`cyrillic\discovered_strings.csv`. При следующем запуске новые строки автоматически
сворачиваются в `cyrillic_strings.csv`. Так словарь дополняется тем, что вы реально
видите в игре.

## ⚙️ Настройки — `cyrillic\config.ini`

При первом запуске создаётся `cyrillic\config.ini` (значения `true`/`false`,
применяются при следующем запуске игры):

```ini
# Автообновление version.dll с GitHub-релизов.
autoupdate=true
# Автосбор показанных строк в discovered_strings.csv (для пополнения словаря).
autocollect=false
# Применять переводы из cyrillic_strings.csv (русский текст в игре).
translate=true
```

| Параметр | По умолчанию | Что делает |
|----------|:---:|------------|
| `autoupdate` | `true` | проверяет и ставит обновления version.dll с GitHub |
| `autocollect` | `false` | собирает показанные строки в `discovered_strings.csv` и сворачивает их в словарь |
| `translate` | `true` | подтягивает `cyrillic_strings.csv` и показывает русский в игре |

## 🔧 Для продвинутых — `gw2-dat-tool` (опционально)

В репозитории есть CLI [`gw2-dat-tool/`](gw2-dat-tool/) для работы со **своим**
`Gw2.dat`: экспорт словаря строк, просмотр `AFNT`-чанков шрифтов, генерация
кириллических атласов. Большинству он **не нужен** — хватает `version.dll` +
`cyrillic-string-dump-tool.exe`. Подробности — в
[`gw2-dat-tool/README.md`](gw2-dat-tool/README.md).

```powershell
# экспорт всего словаря строк в CSV (игра закрыта):
cargo run --release -- --dat "C:\путь\к\Guild Wars 2\Gw2.dat" strs-export-all --out dict.csv
```

## Как это работает (кратко)

Guild Wars 2 хранит шрифты и текст внутри архива `Gw2.dat`. Прямые подходы не
работают: **править `Gw2.dat`** нельзя (клиент проверяет CRC и удаляет изменённые
файлы → перекачка), а **обычный аддон** грузится уже *после* того, как игра
прочитала данные.

`version.dll` обходит оба ограничения: GW2 импортирует её статически, поэтому она
грузится **до** движка; игра читает и проверяет *подлинные* данные, а затем DLL
подменяет **распакованные байты шрифта и строк в ОЗУ** — после всех проверок
целостности, но до парсера. Файл `Gw2.dat` на диске при этом **не изменяется**.

Исходный код инжектируемой `version.dll` закрыт; в релизах публикуется готовый
бинарь.

## ⚠️ Правовая часть / игровые данные

Репозиторий содержит инструмент `gw2-dat-tool` и кириллические данные,
отрисованные из шрифта [Roboto](https://fonts.google.com/specimen/Roboto)
(Apache-2.0). В нём **нет** данных, извлечённых из `Gw2.dat` — это собственность
ArenaNet; готовые файлы вы получаете из *своей* легально установленной копии игры
(через dump-tool) либо из релизов.
